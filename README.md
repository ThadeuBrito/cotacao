# Cotacao


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cotacao'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cotacao

## Usage

    $ cotacao
    #~> Dolar: R$ 3.0380 - Variação: +0.60
    #~> Euro: R$ 3.3450 - Variação: +0.30

    $ cotacao dolar 1000.00
    #~> Dolar: R$ 3.0380 - Variação: +0.60 - Em Reais: R$ 3038.00
